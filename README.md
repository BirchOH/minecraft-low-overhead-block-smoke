# minecraft-low-overhead-block-smoke
This module creates and destroys smoke blocks to achieve a block-based smoke effect. Compared to traditional particle-based or entity-based smoke, it offers substantial performance advantages and reduces server overhead. In entity-sensitive projects, it also helps avoid errors caused by entity conflicts.
The module is based on Python 2.7.18 and uses the Minecraft NetEase/China Edition API (see the API documentation at https://mc.163.com/dev/guide.html). It is packaged as a Python module and can be directly called in script files; refer to the code comments for detailed usage.
